Detailed Description

Contains enums, structures, and function prototypes required for communication between threads and peripherals.

This file includes necessary header files, defines required macros, and declares structures and function prototypes for handling the communication between threads and peripherals. It is mainly used for managing various alerts and the configuration of GPIO pins and serial ports.
