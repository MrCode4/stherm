Detailed Description

Contains constants and default values used in the InputParametrs class for managing Wi-Fi connections.

This file defines constants such as the minimum and maximum number of command-line arguments accepted, the available Wi-Fi management commands, and default values for Wi-Fi manual connection parameters.
