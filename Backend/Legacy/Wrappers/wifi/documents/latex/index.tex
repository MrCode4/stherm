\begin{DoxyItemize}
\item 
\item This program allows users to manage Wi-\/\+Fi connections using the nmcli command-\/line tool.
\item It provides a simple command-\/line interface for searching, connecting, disconnecting,
\item and forgetting Wi-\/\+Fi networks, as well as managing saved connections and retrieving
\item information about the current Wi-\/\+Fi connection.
\item 
\item The program consists of 2 classes\+:
\item -\/ \doxylink{class_input_parametrs}{Input\+Parametrs}\+: Handles the command-\/line arguments and calls the appropriate Wi-\/\+Fi management method.
\item -\/ \doxylink{class_wifi_manager}{Wifi\+Manager}\+: Implements the Wi-\/\+Fi management methods using the nmcli tool.
\item -\/ def\+\_\+values\+: Contains constants and default values used in the \doxylink{class_input_parametrs}{Input\+Parametrs} class for managing Wi-\/\+Fi connections.
\item -\/ main\+: The entry point for the program, which initializes an \doxylink{class_input_parametrs}{Input\+Parametrs} object and calls the execute\+Command() method.
\item 
\item 
\end{DoxyItemize}\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage}

\begin{DoxyItemize}
\item 
\item The program can be run from the command line with the following syntax\+:
\item 
\item wifi\+\_\+manager $<$command$>$ \mbox{[}options\mbox{]}
\item 
\item Where {\ttfamily \texorpdfstring{$<$}{<}command\texorpdfstring{$>$}{>}} is one of the following\+:
\item -\/ {\ttfamily search}\+: Search for available Wi-\/\+Fi networks.
\item -\/ {\ttfamily manual}\+: Manually connect to a Wi-\/\+Fi network using specified parameters.
\item -\/ {\ttfamily connect}\+: Connect to a Wi-\/\+Fi network.
\item -\/ {\ttfamily disconnect}\+: Disconnect from a Wi-\/\+Fi network.
\item -\/ {\ttfamily forget}\+: Remove a saved Wi-\/\+Fi network.
\item -\/ {\ttfamily active}\+: Get MAC of the active Wi-\/\+Fi connection
\item -\/ {\ttfamily connections}\+: Get a list of saved Wi-\/\+Fi connections.
\item -\/ {\ttfamily secret\+\_\+delete\+\_\+all}\+: Remove all saved Wi-\/\+Fi connections.
\item 
\item The {\ttfamily search} command can be run with no options to get a list of available Wi-\/\+Fi networks.
\item The {\ttfamily connect} command requires the SSID and password of the network to connect to.
\item The {\ttfamily manual} command requires the following parameters\+: ESSID, IP address, subnet mask, default gateway, DNS server, DNS server 2, and password.
\item The {\ttfamily disconnect} and {\ttfamily forget} commands require the SSID of the network to disconnect or forget.
